Projects

Messaging app

Me and my teammate designed and developed a messaging application that offers users multiple ways to interact with the platform, including signing up, logging in, or accessing it as a guest. The app enables users to save chats for future reference, providing a seamless and personalized communication experience. It supports both single-person and multi-person chats, with additional options for managing group conversations, such as adding, removing participants, or changing the name of the group. By implementing robust authentication, postman testing, and error handling, I ensured the app was secure and user-friendly, meeting the expectations of a modern communication platform. The project taught me my first steps in creating a full-stack application, integrating front-end and back-end, with the help of my teammate we were able to deliver a product that was both visually appealing and functional.

This project was a significant learning experience, as it required me to integrate front-end and back-end technologies while considering scalability and performance. I focused on creating an intuitive user interface alongside an efficient backend to handle data storage and retrieval. The challenges of implementing real-time communication and data synchronization sharpened my problem-solving and debugging skills. Overall, the project gave me practical insights into building scalable, interactive systems that prioritize user engagement and security.

4-Player Snake Game

The 4-Player Snake Game was a unique and engaging project that combined strategy and programming. In this game, players could code the logic for their snakes to navigate the game board, collect apples, and avoid collisions. The objective was to outscore opponents while ensuring the snake’s survival. This interactive experience encouraged creativity and strategy, allowing players to experiment with different movement algorithms. The project also involved creating a competitive yet fair game environment with clear rules and visually appealing graphics, all while being done with a 2d game.

A major focus of the project was refactoring the codebase to improve its quality and accessibility for future users and developers. By breaking down complex functionalities into smaller, reusable modules, I enhanced the code’s readability and maintainability. I also worked on optimizing performance to ensure smooth gameplay, even with multiple players actively competing. This experience reinforced my understanding of clean coding practices and the importance of designing software that is both efficient and easy to extend or modify.

Text Adventure

The Text Adventure Game is an immersive, command-driven experience where players navigate a building by typing commands to solve puzzles and escape. The game offered a rich storyline with dynamic interactions, allowing players to explore their surroundings and make decisions that influenced the outcome. This project highlighted the power of text-based games in engaging users through creativity and imagination, with no reliance on advanced graphics or interfaces.

My primary contribution was refactoring the game’s code to improve its structure, maintainability, and ease of expansion. I focused on reducing redundancy, simplifying complex logic, and organizing the code into clear, modular components. This effort not only preserved the original gameplay experience but also made the codebase more accessible for future enhancements. The project was an excellent opportunity to refine my ability to work with legacy code while ensuring high-quality standards for future development.
